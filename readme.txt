Wisy Builder is a simple plugin to create awesome pages without any coding knowledge.

== Description ==

Wisy is a free drag-and-drop page builder it will help you to create beautiful pages faster and perfectly.

Note: Wisy support posts & pages at this time.

== Installation ==

Install the plugin and get it working.

**From your WordPress dashboard:**

1. Visit ‘Plugins > Add New’.
2. Search for ‘Wisy’.
3. Activate Wisy from your Plugins page.

**From WordPress.org:**

1. Download Wisy.
2. Upload the ‘wisy’ directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)
3. Activate Wisy from your Plugins page.

== How To Use It ==

1. Create new post.
2. Go back to Posts List.
3. Search for the post that you've created, and click "Edit With Wisy".
